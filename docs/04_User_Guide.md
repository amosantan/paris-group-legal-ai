## 3. Features

This section provides detailed instructions on how to use each of the key features of the platform.

### 3.1. New Consultation

To start a new legal consultation:

1. Click on **New Consultation** in the navigation menu.
2. Fill in the **Title**, **Category**, and **Language** for the consultation.
3. Click **Create Consultation**.
4. You will be redirected to the consultation page, where you can start chatting with the AI assistant.

### 3.2. Document Generator

To generate a legal document:

1. Click on **Document Generator** in the navigation menu.
2. Select the type of document you want to generate (Demand Letter, Eviction Notice, NOC, or LOI).
3. Fill in the required information in the form.
4. Click **Generate** to create the document as a PDF.

### 3.3. Legal Translator

To translate a legal document:

1. Click on **Legal Translator** in the navigation menu.
2. Select the **Source Language** and **Target Language**.
3. Enter the text you want to translate in the input box.
4. Click **Translate** to get the professional legal translation.

### 3.4. Knowledge Base

The Knowledge Base is a repository of legal documents and articles. You can browse the knowledge base to find relevant information for your cases.

### 3.5. PDF Upload

The PDF Upload feature allows you to upload and process PDF documents. This feature is still under development.

### 3.6. LLM Settings

The LLM Settings page allows you to configure the LLM provider for the platform. You can choose between Google Gemini and the Manus Built-in LLM.
